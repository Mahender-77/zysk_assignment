# Zysk_Assignment


## Tech Stack

- React
- Chakra UI


## Getting Started

### Installation

1. **Clone the repository:**

   ```shell
   git clone https://github.com/Mahender-77/zysk_assignment.git

   ```

2. **Install Dependencies:**

   ```shell
   cd zysk_assignment
   npm install
   
   ```

3. **Start the server**

   ``` 
   npm start 
   ```
